# Neovim

Personal Neovim configuration using `vim.pack` as plugin manager and [Catppuccin Macchiato](https://github.com/catppuccin/nvim) as colorscheme.

## Plugins

| Category | Plugin(s) |
|----------|-----------|
| Colorscheme | [catppuccin/nvim](https://github.com/catppuccin/nvim) |
| Completion | [blink.cmp](https://github.com/saghen/blink.cmp), [friendly-snippets](https://github.com/rafamadriz/friendly-snippets) |
| AI | [opencode.nvim](https://github.com/nickjvandyke/opencode.nvim) |
| Database | [vim-dadbod](https://github.com/tpope/vim-dadbod), [vim-dadbod-ui](https://github.com/kristijanhusak/vim-dadbod-ui), [vim-dadbod-completion](https://github.com/kristijanhusak/vim-dadbod-completion), [vim-dadbod-ssh](https://github.com/pbogut/vim-dadbod-ssh) |
| Editing | [mini.ai](https://github.com/echasnovski/mini.nvim), [mini.comment](https://github.com/echasnovski/mini.nvim), [mini.icons](https://github.com/echasnovski/mini.nvim), [mini.pairs](https://github.com/echasnovski/mini.nvim), [mini.splitjoin](https://github.com/echasnovski/mini.nvim), [mini.surround](https://github.com/echasnovski/mini.nvim), [mini.trailspace](https://github.com/echasnovski/mini.nvim) |
| Git | [fugitive](https://github.com/tpope/vim-fugitive), [gitsigns](https://github.com/lewis6991/gitsigns.nvim), [diffview](https://github.com/sindrets/diffview.nvim) |
| HTTP | [kulala.nvim](https://github.com/mistweaverco/kulala.nvim) |
| Markdown | [render-markdown.nvim](https://github.com/MeanderingProgrammer/render-markdown.nvim) |
| LSP | [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig), [mason.nvim](https://github.com/mason-org/mason.nvim) |
| Navigation | [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim), [oil.nvim](https://github.com/stevearc/oil.nvim) |
| Treesitter | [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter), [nvim-treesitter-textobjects](https://github.com/nvim-treesitter/nvim-treesitter-textobjects) |
| UI | [noice.nvim](https://github.com/folke/noice.nvim), [lualine.nvim](https://github.com/nvim-lualine/lualine.nvim), [which-key.nvim](https://github.com/folke/which-key.nvim), [nvim-highlight-colors](https://github.com/brenoprata10/nvim-highlight-colors) |
